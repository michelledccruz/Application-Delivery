# Final Calculator Application

This project is a Java calculator application that performs basic math operations and memory functions. The program was created for the 5.2 Application Delivery project.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exception handling
- Store and retrieve single memory values
- Clear memory values
- Store integers in a memory list
- Display memory list
- Calculate sum and average of memory list
- Display difference of first and last values
